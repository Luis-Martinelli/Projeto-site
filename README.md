🌐 Projeto de Site Organizacional

Site institucional desenvolvido para centralizar informações, facilitar o acesso a sistemas e apresentar dados organizacionais de forma clara e intuitiva.

📸 Preview


/assets/preview.png

📌 Funcionalidades
📢 Banner informativo com destaques importantes
⚡ Acesso rápido às aplicações mais utilizadas
🏆 Destaque para funcionário do mês
🏢 Apresentação da área de atuação
🖼️ Galeria de fotos institucional
📍 Listagem de endereços das unidades subordinadas
🛠️ Tecnologias Utilizadas
💻 HTML5
🎨 CSS3
⚙️ JavaScript
✨ Plugins de animação (fade-in)
✨ Plugins lightbox2 (galeria de fotos)
