🌐 Projeto de Site Organizacional


Site institucional desenvolvido para centralizar informações, facilitar o acesso a sistemas e apresentar dados organizacionais de forma clara e intuitiva.

📌 Funcionalidades <br>
📢 Banner informativo com destaques importantes <br>
⚡ Acesso rápido às aplicações mais utilizadas <br>
🏆 Destaque para funcionário do mês <br>

📍 Listagem de endereços das unidades subordinadas <br> <br>

⚙️ JavaScript <br>
